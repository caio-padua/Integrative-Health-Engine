FASE 2 — ROTAS NOVAS
Arquivo: artifacts/api-server/src/routes/cavaloClinical.ts
═══════════════════════════════════════

ENTRADA OPERACIONAL (equipe da clínica):
POST /entrada/operacional/visita
  → Salva dados_visita_clinica
  → Se AMARELO: cria task_card para ENFERMEIRA02
  → Se VERMELHO: cria task_card PRIORITARIO + notifica médico

POST /entrada/operacional/exames
  → Salva arquivos_exames com origem=OPERACIONAL
  → Organiza no Google Drive: /Pacientes/[Nome]/Exames/Recebidos/

POST /entrada/operacional/exames-valores
  → Salva exames_evolucao com classificação por terços
  → Ativa motor clínico automaticamente

ENTRADA AUTÔNOMA (paciente externo via site):
POST /entrada/autonoma/checkin
  → Salva com status AGUARDANDO_VALIDACAO
  → origem=AUTONOMA
  → Verifica cascata_validacao_config.ativa
  → Se cascata ATIVA: cria validacoes_cascata na sequência
  → Se cascata INATIVA: ativa motor diretamente após 1 aprovação

POST /entrada/autonoma/upload-exame
  → Recebe arquivo via multipart/form-data
  → Salva arquivos_exames com origem=AUTONOMA
  → Nomeia automaticamente: AAAA.MM.DD [TIPO] [AREA ou NOME]

VALIDAÇÃO DA CASCATA:
PATCH /validacao/:validacaoId/aprovar
  → Requer autenticação por role
  → Registra assinatura A1 se configurado
  → Avança para próxima etapa da cascata
  → Quando última etapa aprovada: ativa motor clínico

ADMIN:
GET  /admin/cascata/config
PATCH /admin/cascata/config
  → Apenas médico senior pode alterar
  → Liga/desliga cada etapa individualmente

LÓGICA DOS TERÇOS (função reutilizável):
function classificarExame(valor, minimo, maximo):
  if valor < minimo → "PREOCUPANTE"
  if valor > maximo → "ALERTA"
  terco = (maximo - minimo) / 3
  if valor <= minimo + terco → "BAIXO"
  if valor <= minimo + (2 * terco) → "MEDIANO"
  else → "ÓTIMO"

HIERARQUIA DE NOTIFICAÇÕES:
VERDE → histórico + PDF evolutivo (sem card)
AMARELO → card ENFERMEIRA (prioridade ALTA)
         + card MEDICO_ASSISTENTE (prioridade MEDIA)
         + card DR_CAIO (prioridade BAIXA)
VERMELHO → card PRIORITARIO todos os níveis
          + notificação imediata médico responsável
